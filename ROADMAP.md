# Inspired Mile — Project Roadmap

## Overview

Development plan from concept to live platform, organized into 5 phases.
Team commitment target: 3–5 hours per week per person.

---

**Goal**: Legal entity + project infrastructure

- [x] Voyage Tracable, LLC established (DBA Inspired Mile) — Sarasota, FL
- [ ] Complete all member paperwork and operating agreement
- [ ] Set up Inspired Mile email accounts for all team members
- [ ] Set up GitHub organization and repositories
- [ ] Set up project management (GitHub Projects or Linear)
- [ ] Apply for Meta Horizon Start developer program

## Phase 1: Webapp Prototype (Weeks 4–10)
**Goal**: Working digital card collection + AI battle experience demo

### 1A: Core Webapp + Playground
- [ ] Landing page with project vision and card concept showcase
- [ ] Digital card collection browser with owned/locked states and filters
- [ ] Card detail view with animated/3D preview
- [ ] Playground page: card acquisition hub (packs, trivia, challenges, battle rewards)
- [ ] Inventory system with localStorage persistence
- [ ] Responsive mobile-first design

### 1B: AI Master Thespian (v1)
- [ ] Define Master Thespian persona and system prompt
- [ ] Integrate WebLLM (Llama 3.2 3B Q4, in-browser via WebGPU)
- [ ] Build loading UI with model download progress
- [ ] Build chat/battle interface for text-based rap battles
- [ ] Integrate Kokoro TTS (82M, in-browser) for AI voice output
- [ ] Add microphone input via Web Speech API for voice battles
- [ ] Create 5–10 sample Shakespeare battle prompts
- [ ] Add scoring/ranking system (basic)
- [ ] Test on Chrome, Edge (WebGPU required)

### 1C: Initial Card Art
- [ ] Design digital card template
- [ ] Create 10 character cards across 3 Shakespeare houses
- [ ] Create 5 relic/potion item cards
- [ ] Generate AI-assisted artwork for each card
- [ ] Design Sonnet Man special tier card

**Deliverable**: Playable webapp demo, 15+ card designs, AI battle prototype
